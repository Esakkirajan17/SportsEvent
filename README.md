A professional **Sports Event Management System** developed using **Python Flask** and **SQLite** to manage tournaments, teams, players, schedules, venues, and event registrations efficiently.

---

## 📌 Project Overview

This system is designed for colleges, schools, sports clubs, and organizations to automate the management of sports events and tournaments.

It reduces manual work by providing a centralized platform for:

- Event Creation
- Team Registration
- Player Management
- Match Scheduling
- Venue Allocation
- Statistics Tracking

---

## ✨ Features

## 👤 User Module

- User Registration
- User Login
- Team Registration
- Add Players
- View Sports Events
- Dashboard Access

## 🔐 Admin Module

- Secure Admin Login
- Create Sports Events
- Manage Teams
- Manage Players
- Manage Venues
- Generate Match Schedules
- View Statistics
- Venue Availability Management
